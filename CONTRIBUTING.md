# Contributing to VAJRA Framework

First of all, thanks for taking the time to contribute to **VAJRA**, the God-Tier Cyber Security Framework! Created by Chandan Pandey.

## 🧠 Philosophy
We are building a tool for **2026 and beyond**. If your contribution doesn't involve AI, Quantum computing, Web3, or extreme asynchronous speeds, it might not fit the vision. Think BIG. Think God-Tier.

## 🛠️ Development Workflow

1. **Fork the Repo**: Fork `vajra-terminal` or `vajra-brain`.
2. **Branch**: Create a feature branch (`git checkout -b feature/quantum-fuzzing`).
3. **Rust Standards**: 
    - Use `cargo clippy` to ensure your Rust code is extremely optimized.
    - Never use blocking synchronous network calls. Always use `tokio` and `reqwest`.
4. **Commit**: Keep commits clean (`feat: added new polymorphic generator`).
5. **PR**: Open a Pull Request. Chandan Pandey or core maintainers will review it.

## 🐞 Bug Reports
If the UI freezes or the Matrix rain stops, report it immediately! Provide the exact OS (Kali/Arch/Parrot) and the Rust backtrace.
