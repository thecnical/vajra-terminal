# Security Policy

## Supported Versions
Currently, only **v1.0.0 (God-Tier Edition)** is actively supported with security updates.

## Reporting a Vulnerability

**VAJRA** is a security tool. Ironically, if you find a vulnerability within the tool itself (e.g., a buffer overflow in the Rust TUI or an SSRF in the Python Brain), please report it responsibly.

1. Do NOT open a public GitHub issue for critical security vulnerabilities.
2. Email the core team securely.
3. Provide a Proof of Concept (PoC).

We will patch the framework within 24 hours via the P2P Swarm Network update protocol.
